# 🛒 E-Commerce Website Frontend (TechnoHack Internship)

## 📌 Overview
This project demonstrates a complete **frontend development workflow** for a modern E-commerce website using core web technologies.  
It includes responsive UI design, product listing, cart functionality, and interactive user interface features.

The project simulates a real-world online shopping platform and is structured to be easily extendable with backend integration in the future.

---

## 🎯 Objectives
- Build a modern and responsive E-commerce UI  
- Implement dynamic frontend functionality using JavaScript  
- Create an interactive shopping experience  
- Manage cart operations using localStorage  
- Prepare project for backend/API integration  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)

## 🔍 Project Features

* 🏠 Responsive Home Page with modern UI
* 🛍️ Product listing with cards and details
* 🛒 Add to cart / remove from cart functionality
* 🔢 Quantity update and total price calculation
* 💾 Cart data stored using localStorage
* 🔐 Login & Register UI (frontend only)
* 📱 Fully responsive design
